Що таке Blender?

Це пакет для створення тривимірної комп'ютерної графіки. Її використовують для 3D моделювання та візуалізації - вони потрібні для різних сфер діяльності. Насамперед вони потрібні для 3D модельєрів — вони створюють 3D моделі персонажів, будівель, техніки тварин для ігор, і рідше в кіноіндустрії.

Що можна зробити у Blender?

Modeling

- Modeling - процес створення, зміни тривимірної моделі об'єкта.

Sculpting

- Sculpting - режим аналогічний режиму редагування тому, що він використовується зміни форми моделі, але у режимі скульптингу використовується зовсім інший робочий процес: замість роботи з окремими елементами (вершинами, ребрами і гранями) область моделі змінюється з допомогою пензля.

3D/2D Animation & Rendering

- Animation - режим зміни параметрів об'єктів у часі.

- Rendering - це процес перетворення 3D-сцени на 2D-зображення. Blender включає три движки рендеринга з різними можливостями:

• Eevee - це фізично коректний рендеринг реального часу.

• Cycles - це фізично коректний трасувальник шляху.

• Workbench призначений для компонування, моделювання та попереднього перегляду.
